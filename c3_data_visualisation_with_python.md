# Data Visualization with python

## Matplotlib

Matplotlib is a static plotting library for python.

## Plotly

Plotly is a dynamic plotting library for python. 

## Dash

Dash is a web application framework for python. It is used to create interactive web applications. It is built on top of Flask, Plotly, and React.js. It is used to create dashboards and data visualization applications.

## Streamlit

Streamlit is a web application framework for python. It is used to create interactive web applications. It is built on top of Flask and React.js. It is used to create dashboards and data visualization applications.

## Tableau

Tableau is a data visualization tool that allows users to create interactive and shareable dashboards. It is a powerful tool for data analysis and visualization, and it is widely used in the industry. Tableau provides a drag-and-drop interface that makes it easy to create complex visualizations without needing to write code. It also supports a wide range of data sources, making it versatile for various data analysis tasks.

## Amazon QuickSight

Amazon QuickSight is a cloud-based business intelligence service provided by Amazon Web Services (AWS). It allows users to create and publish interactive dashboards and visualizations. QuickSight is designed to be fast and scalable, making it suitable for analyzing large datasets. It integrates seamlessly with other AWS services, such as S3, Redshift, and RDS, allowing users to easily connect to their data sources. QuickSight also offers features like machine learning insights and natural language querying, making it a powerful tool for data analysis and visualization in the cloud.
